# LumiAds — Google Stitch 디자인 프롬프트

## 프로젝트 개요

LumiAds는 AI 기반 광고 자동화 SaaS입니다. 네이버/메타/구글 3대 광고 플랫폼을 통합 관리하며, 4개 AI 에이전트가 토론(디베이트)하여 최적 광고 전략을 도출하고, 사람이 승인하면 API로 자동 등록·최적화합니다.

### 디자인 톤앤매너
- **스타일**: 클린, 모던, 프로페셔널 SaaS 대시보드
- **컬러**: Primary Blue (#3B82F6), 흰색 배경, 연한 회색 보조 배경
- **폰트**: Pretendard (한글), Inter (영문)
- **아이콘**: Lucide Icons
- **컴포넌트**: shadcn/ui 스타일 (Tailwind CSS 기반, 둥근 모서리, 미니멀)
- **레이아웃**: 좌측 사이드바 + 상단 헤더 + 메인 콘텐츠 (고정 레이아웃)
- **언어**: 한국어 (UI 텍스트 전부 한국어)
- **반응형**: 데스크톱 우선 (1440px 기준), 모바일 대응

| # | 페이지 | 경로 | 설명 |
|---|--------|------|------|
| 1 | 로그인 | `/login` | 이메일/비밀번호 로그인 |

| 7 | 소재 관리 | `/creative` | 소재 갤러리 (그리드) + 상세 모달 |

| 12 | 고객 관리 | `/settings/tenants` | 광고주 목록 + 등록 + 사용량 |
